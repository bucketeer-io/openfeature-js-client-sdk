# React Native Example

For React Native instructions, please see our [React Native documentation](./../../docs/react-native.md)

## Setup

1. Copy `.env.example` to `.env`.
   ```bash
   cp .env.example .env
   ```
2. Update `.env` with your Bucketeer secrets.
3. Install dependencies:
   ```bash
   pnpm install
   ```
4. Setup the native dependencies:
   ```bash
   npx expo prebuild
   ```

5. Start the app:
   ```bash
   pnpm start
   ```
