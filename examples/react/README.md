# React Example

For React instructions, please see our [React documentation](./../../docs/react.md)

## Setup

1. Copy `.env.example` to `.env`.
   ```bash
   cp .env.example .env
   ```
2. Update `.env` with your Bucketeer secrets.
3. Install dependencies:
   ```bash
   pnpm install
   ```
4. Start the development server:
   ```bash
   pnpm dev
   ```

If you want to try it on iOS and Android, please make sure to run `npx expo prebuild` to set up the iOS and Android projects before running the app.
